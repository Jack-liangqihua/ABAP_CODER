更新本地信息到远程仓库步骤说明：

1、git add '文件名称'

2、git commit -m '更新说明'

3、git push  ABAP_CODER master . 

++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

更新远程信息到本地仓库步骤说明：

1、git remote -v                      查看远程分支

2、git fetch ABAP_CODER master:temp   从远程获取最新版本到本地

3、git diff temp                      比较本地仓库与下载的temp分支

4、git merge temp                     合并temp分支到本地的master分支

5、git branch -d temp                 删除temp分支
